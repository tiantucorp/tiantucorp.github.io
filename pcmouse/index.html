PC Mouse

Turn Android phone into powerful PC Mouse, Keyboard, Media Controller, and Presentation Controller for Windows.

If any of these sound like you:

  • You want a smooth WiFi touchpad to control your PC from across the room

  • You wish your PC had voice typing or handwriting input like your Android phone

  • You want to control videos or music while relaxing a few feet away from your TV

  • You present ideas and want to move freely, while advancing slides from anywhere in the room

Then PC Mouse is built for you.

Step 1) Install Android App from Google Play Store or from Samsung Galaxy Store. How to connect:
- Scan QR code
- or enter IP and port
- Make sure service of Windows App is reachable

Step 2) Install Windows App (2.2 or above) from Microsoft Store or Install Windows App in PC.

And for better performance, please

Step 3) Install Windows Suite (2.2 or above) from Microsoft Store or Install Windows Suite in PC.
