PC GPS

Turn Android phone into PC GPS dongle for Windows, streaming NMEA 0183 data over virtual COM or TCP/IP.

Step 1) (Install Android App from Google Play Store,or from Samsung Galaxy Store.) How to connect:
- Scan QR code
- or enter IP and port
- Make sure service of Windows Suite is reachable

Step 2) Install Windows PCSpeaker Suite(2.3 and above) in PC from Microsoft Store. or Install Windows Suite in PC.

In the Android App, after you scan or enter IP, it will get connected, and start streaming NMEA 0183 data. In the Windows, once connected, you could open Google Earth Pro or OpenCPN to test over virtual COM or TCP/IP.

Warning: if you are only able to do OpenCPN, it means that you might have installed Windows PCSpeaker App in PC. Please install Windows PCSpeaker Suite(2.3 and above) in PC.
