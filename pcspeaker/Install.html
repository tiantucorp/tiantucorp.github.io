How to Install This APK on Your Android Device

Follow these simple steps to install the app directly from this website.

1. Download the APK file

Download APK

Your browser may ask to confirm the download. Choose Allow or Download.

2. Enable installation from your browser

Android blocks unknown-source installs by default. To continue:

  • When prompted, tap Settings.
  • Enable Allow from this source.
  • Return to your browser.

(You only need to do this once.)

3. Open the downloaded file

  • Swipe down from the top of your screen.
  • Tap the downloaded file named app-release.apk.
  • Or open it from Files ? Downloads.

4. Install the app

  • Tap Install.
  • Wait a few seconds.
  • Tap Open to launch the app.

5. Updates

Since this APK is installed from the website, updates will not come from Google Play or Galaxy Store. When a new version is available, simply download and install the updated APK the same way.
