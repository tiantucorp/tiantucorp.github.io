PC Speaker

Turn Android phone into PC Speaker, Mic, and Camera for Windows.

Step 1) (Install Android App from Google Play Store, from Samsung Galaxy Store or from this website.) How to connect:
- Enter IP and port
- Or scan QR code
- Make sure service of Windows App is reachable

Step 2) Install Windows App from Microsoft Store or Install Windows App in PC.

Test Speaker, Mic and Camera function before install Windows Driver(Test version).

In the Windows App's Dashboard, click on "Capture Tab Audio" Button. In the Android App, click on Speaker->Start Speaker->Connect to Remote Speaker, and you will hear the audio stream from PC Browser Tab. 

Step 3) Install Windows Driver(Test version) from Microsoft Store or Install Windows Driver(Test version) in PC. Finally turn your Android phone into PC Speaker, Mic, and Camera for Windows.
